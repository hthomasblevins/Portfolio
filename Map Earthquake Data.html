Map Earthquake
Data

This project sources JSON-encoded, cloud-based data, extracts and summarizes data using Power Query and Power BI methods.

1. Extact JSON data from the USGS, transform and load using Power Query.

After extracting the earthquake data, I transformed the data into columns and rows in the proper format for lattitude, longitude, dates, etc.

  • Link to Power Query Code


2. Map earthquakes that occurred in the last 30 days.

After prepping the data, I used Power BI to generate a map with bubbles representing earthquakes with the size indicating magnitude and color representing the # of people reporting that the felt the earthquake. There are also 'slicers' or filters on the left that are limiting the display to earthquakes that were felt by at least 100 people and having a magnitude of 3.00 or higher. I also included some summary statistics in the bottom left.
